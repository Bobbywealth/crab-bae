# Crabbae - Bold Seafood, Big Energy

A Next.js website for Crabbae, a seafood restaurant with locations in Kenilworth and Maplewood, New Jersey.

## Features

- Responsive design with Tailwind CSS
- Dark theme with red/orange gradient accents
- Hero section with background image and logo
- Location cards for both Kenilworth and Maplewood
- Featured dishes menu section
- Photo gallery
- About section featuring founder Kashif Jones
- Reservation section (links to OpenTable)
- Contact form
- Mobile-friendly navigation

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the result.

## Tech Stack

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
